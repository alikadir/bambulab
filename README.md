### Filament Temperature Formula
((MinTemp * 3) + (MaxTemp * 2)) / 5 = OptimalTemp
```
((190 * 3) + (220 * 2)) / 5‎ = 202
((190 * 3) + (230 * 2)) / 5‎ = 206
((195 * 3) + (220 * 2)) / 5‎ = 205
((205 * 3) + (225 * 2)) / 5‎ = 213
((210 * 3) + (230 * 2)) / 5‎ = 218
```
### Presets
0.4mm Nozzle = 0.16mm Optimal
